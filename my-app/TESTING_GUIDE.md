## 🐛 Если что-то не работает

### Проблема: 404 на странице профиля

**Решение:**
```
1. Проверьте, что вы авторизованы
2. Откройте /login и логируйтесь заново
3. Проверьте в Firebase Console что пользователь есть в коллекции users
```

---

### Проблема: Не вижу админ-панель

**Решение:**
```
1. Проверьте в Firebase Console: isAdmin должен быть true
2. Обновите страницу (Ctrl+F5 для очистки кэша)
3. Проверьте консоль браузера (F12) на ошибки
4. Логируйтесь заново: /logout → /login
```

---

### Проблема: "Permission denied" ошибки

**Решение:**
```
1. Откройте Firebase Console → Firestore → Rules
2. Проверьте, что правила опубликованы
3. Убедитесь, что структура документов соответствует правилам
4. Проверьте, что вы авторизованы в Firebase Auth
```

---

### Проблема: Данные не обновляются

**Решение:**
```
1. Проверьте консоль браузера на ошибки
2. Обновите страницу (F5)
3. Проверьте, что у вас есть права на редактирование в Firestore Rules
4. Проверьте подключение к интернету
```
