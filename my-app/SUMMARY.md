## 🚀 Как начать использовать

### Шаг 1: Применить Firestore Rules
```
1. Firebase Console → Firestore Database → Rules
2. Скопировать firestore.rules.txt
3. Опубликовать
```

### Шаг 2: Создать администратора
```
1. Зарегистрируйтесь в приложении
2. Firebase Console → Firestore → Collection users
3. Установить isAdmin: true
```

### Шаг 3: Протестировать
```bash
npm run dev
# Открыть http://localhost:5173
# Попробовать: /profile, /admin, /logout
```

## 🔐 Безопасность

### Что защищено:
- ✅ Пользователи видят только свои профили
- ✅ Администраторы видят все профили
- ✅ Только администраторы могут изменять товары
- ✅ Заказы защищены от несанкционированного доступа
- ✅ Корзины видны только их владельцам

### Firestore Rules покрывают:
- 🛡️ Аутентификацию
- 🛡️ Авторизацию (проверка ролей)
- 🛡️ Контроль доступа данных
- 🛡️ Защиту целостности данных

## 💻 API контекста аутентификации

### useAuth() hook

```typescript
const {
  user,                    // User | null - текущий пользователь
  firebaseUser,           // FirebaseUser | null - Firebase User
  isLoading,              // boolean - загружается ли
  register,               // функция регистрации
  login,                  // функция входа
  logout,                 // функция выхода
  resetPassword,          // функция восстановления пароля
  updateUserProfile,      // функция обновления профиля
  updateUserPassword,     // функция изменения пароля
} = useAuth();
```
